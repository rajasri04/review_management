Project Title: REV-SYS: Streamlined Project Management Platform

Description:

Developed a web-based application (REV-SYS) to enhance student project management within a college environment.
Improved efficiency by automating report maintenance, management, and review processes.
Facilitated real-time communication by enabling instant sharing of project updates and feedback among students, faculty guides, and administrators.
Increased transparency through real-time access to attendance and progress reports.
